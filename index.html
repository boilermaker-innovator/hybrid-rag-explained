Jon's Assistant — Setup

Fill in the fields below, then follow the steps to get running

1
Claude API Key
Get your key from console.anthropic.com/settings/keys. Click "Create Key", give it a name, and paste it below.
2
Settings (optional)
Defaults work fine — only change these if you know what you're doing.
Default is 3000. Change if something else uses that port.
claude-sonnet-4-20250514 (fast+cheap) or claude-opus-4-20250514 (smarter+pricier)
3
Generate your .env file
Click the button to generate your config. Then copy the contents into a new file called .env in your project folder.
How to create the file:
1. Open Notepad
2. Paste the contents above
3. File → Save As → navigate to your jon-assistant folder
4. Set "Save as type" to All Files (*.*)
5. Name the file exactly: .env
6. Click Save
4
Install and Run
Make sure you have Node.js installed (LTS version). Then open a terminal in your project folder and run:
> npm install
> npm start
To open a terminal in your folder: Open File Explorer, navigate to jon-assistant, click the address bar, type cmd and hit Enter.
5
Verify it works
Open http://localhost:3000 in your browser. You should see the chat interface.
  • Downloaded all project files
  • Created .env with API key
  • Ran npm install
  • Ran npm start
  • Chat loads in browser
🔒 Your API key stays on your machine. Never share it in chats, emails, or commit it to GitHub.
